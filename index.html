Bem-vindo ao meu site de exemplo Flexbox

Este site utiliza um arquivo CSS separado, para criar o menu horizontal com Flexbox, além de uma aparência limpa e moderna

as tags exigidas estão todas presentes: <header>, <main>, <div>,(Container interno) <h1> entre outros.

O menu horizontal é obtido com display: flex na lista <ul>. O CSS externo facilita a manutenção e o aprendizado separado da estrutura e da apresentação.

Agora o HTML cuida do conteúdo e o CSS do visual. Basta salvar dois arquivos e direcionar o link para o arquivo correto.
